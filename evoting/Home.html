  About Us

♦ The purpose of E-Voting System is to facilitate online web based automated system by the help of computerized and full-fledged computer software. fulfilling the organization’s requirements, so that their valuable data/information can be stored for a longer period with easy accessing and manipulation of the same.
♦ To provide facility of managing the voters and contesting candidate’s information.
♦ To provide the facility to admin to schedule the election which the organization has to conduct for their current requirement.
♦ To provide the facility of username and password to admin for data security.
♦ To provide validation to voting process to ensure that only eligible voters which are registered on the system can vote.
♦ To provide the facility of username and password to voters for their secure voting process.
♦ To generate different reports of voters details, contesting candidates details and voting reports which can be accessed by only admin.
♦ To provide a facility to admin to access voting data and manage candidates details.
♦ To facilitate the feature of single vote from single candidate to avoid the double or repeated voting by voter for same election.
♦ To generate results by calculations which can be access by the admin.

  Help

The system “E-voting System” is a web application that is used for maintain the records in an organized manner and to replace old paper work system.
E-voting is automatic electronic system to aid casting and counting votes.
Updating and modifications will be easily achievable and all the calculations and counting work would be accurate.
The E-Voting system will manage the voter’s information by which voter can login and use his voting rights.
Candidates which are shortlisted have to register to system to obtain for election.
The system will corporate all features of voting system. It provides the voters vote for every party.
System uses database which is maintained by admin with all the names of voter’s candidates and parties with their complete information.
Voting details are store in database and result is displayed by the calculation.
Web application means that the system will be available for access 24/7 except when there is a temporary server issue which is expected to be minimal.
Organization admin as well as the company’s staff will be able to use the system effectively.
System can be only accessible for administrative staff and organization staff.
