         The system “E-voting System” is a web application that is used for maintain the records in an organized manner and to replace old paper work system. E-voting is automatic electronic system to aid casting and counting votes. Updating and modifications will be easily achievable and all the calculations and counting work would be accurate. The E-Voting system will manage the voter’s information by which voter can login and use his voting rights. Candidates which are shortlisted have to register to system to obtain for election. The system will corporate all features of voting system. It provides the voters vote for every party. System uses database which is maintained by admin with all the names of voter’s candidates and parties with their complete information. Voting details are store in database and result is displayed by the calculation. Web application means that the system will be available for access 24/7 except when there is a temporary server issue which is expected to be minimal. Organization admin as well as the company’s staff will be able to use the system effectively. System can be only accessible for administrative staff and organization staff.
